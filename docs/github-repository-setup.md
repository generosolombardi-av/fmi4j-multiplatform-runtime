## First Release

Tag:

```text
v0.38.0-multiplatform.1
```

Release title:

```text
Initial multiplatform runtime fork
```

Release notes:

```text
This is the first public release of the FMI4j Multiplatform Runtime community fork.

Highlights:
- Preserves the original FMI4j import API.
- Adds a macOS universal native runtime for FMI4j import.
- Supports both arm64 and x86_64 in native/fmi/darwin64/fmi4j-import.dylib.
- Keeps existing Windows and Linux native runtime resources.
- Adds documentation for native runtime builds.
- Adds scripts for building and packaging the macOS universal runtime.
- Adds roadmap and contribution notes for future multiplatform support.

Maintainer:
- Generoso Lombardi, creator and technical director of the SinergIA project.
```
